Opening Files

To open files with lots oflines follow these steps. 1.Click file, and then a new panel will slide in. 2. In new panel click open. 3. This will then proceed to open a file browswer. 4. In the file browswer, the top row will represent the current file path directory. and the main window will represent the files in the directory. 5. Proceed to double click on any csv file to load it into the program and the 4 visualizations.

Choosing betweeen Visualizations

To choose a particular visualization, follow these instructions. 1. Once you have opened a visualization, the collocated lines visualization will automatically be selected. 2. On the left panel, you will see a button, the third from the bottom that says visualization, proceed to click button. 3. The main window will then be split into 4 sections, each section , representing a different visualization, top left being collocated, top right being radial paired, bottom left being shifted paired, and the bottom right being inline diminesions. 4. Double click any of the 4 sections to view that corresponding visualization. 5. To choose another visualization, repeat from step 2.

Viewing the dataset

To view the dataset currently being visualized following these steps steps. 1.After the dataset has been loaded, click the button on the left that says show table, and a data table with all the vectors will pop up from the right. 2. Now when you hover over a vector in the table, that corresponding vector will light up in the visualization. 3. In addition that vector that is hovered over will have dimension values displayed at top of the tabled, underneath where it says Data Table. 4. To hide the table, simply click the button that says hides tabled on the main panel on the left.

Animating vectors

Lots of Lines actually has a very robust animation system. 1. To animate lines simply clck the line you want animated, when you click the line, the line should then start flashing. 2. To animation that line simply click the animate button on the left Panel.

Animating Multiple Vecotrs

Lots of Lines actually has a very robust animation system that will also allow you to animate multiple lines. 1. click and drag through the lines you want animated and left go. You should then the see the selected lines flash. 2. To animation the lines simply click the animate button on the left Panel.

Changing the animation speed

Lots of Lines actually has a very robust animation system that will also allow you change the speed of animations. 1. On the main panel on the left click the left gear icon the bottom left quarter. 2. This will then bring a settings dialog to the center of the screen. 3. Under Animation duration, type in a value for how long you want the animations to last, and click okay. 3. Now proceed to select and animate lines, and the speed of the animation will now reflect your choice in the settings panel.
