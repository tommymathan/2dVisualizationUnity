We are Lots of Lines

What is Lots of Lines? Lots of Lines is a data visualization application designed to display multidimensional data in a lossless format. Individuals can push spreadsheet data to the visualization application, which will draw the provided data using 4 visualization methods. Users can then interact with the provided dataset by coloring subsets of interest, hiding outliers and animating vectors as a function of time. Lots of Lines provides a toolset which users may explore their data, quickly discovering and tagging trends and outliers.

Opening Files

To open files with lots oflines follow these steps. 1.Click file, and then a new panel will slide in. 2. In new panel click open. 3. This will then proceed to open a file browswer. 4. In the file browswer, the top row will represent the current file path directory. and the main window will represent the files in the directory. 5. Proceed to double click on any csv file to load it into the program and the 4 visualizations.

Choosing betweeen Visualizations

To choose a particular visualization, follow these instructions. 1. Once you have opened a visualization, the collocated lines visualization will automatically be selected. 2. On the left panel, you will see a button, the third from the bottom that says visualization, proceed to click button. 3. The main window will then be split into 4 sections, each section , representing a different visualization, top left being collocated, top right being radial paired, bottom left being shifted paired, and the bottom right being inline diminesions. 4. Double click any of the 4 sections to view that corresponding visualization. 5. To choose another visualization, repeat from step 2.

Viewing the dataset

To view the dataset currently being visualized following these steps steps. 1.After the dataset has been loaded, click the button on the left that says show table, and a data table with all the vectors will pop up from the right. 2. Now when you hover over a vector in the table, that corresponding vector will light up in the visualization. 3. In addition that vector that is hovered over will have dimension values displayed at top of the tabled, underneath where it says Data Table. 4. To hide the table, simply click the button that says hides tabled on the main panel on the left.

Animating vectors

1. To animate lines simply click the line you want animated, when you click the line, the line should then start flashing. 2. To animation that line simply click the animate button on the left Panel.

Animating Multiple vectors

Lots of Lines actually has a very robust animation system that will also allow you to animate multiple lines. 1. click and drag through the lines you want animated and left go. You should then the see the selected lines flash. 2. To animation the lines simply click the animate button on the left Panel.

Changing the animation speed

Lots of Lines actually has a very robust animation system that will also allow you change the speed of animations. 1. On the main panel on the left click the left gear icon the bottom left quarter. 2. This will then bring a settings dialog to the center of the screen. 3. Under Animation duration, type in a value for how long you want the animations to last, and click okay. 3. Now proceed to select and animate lines, and the speed of the animation will now reflect your choice in the settings panel.

Manipulating Lines

Removing lines from the visualization

1. To remove a line click a line until it flashes, or drag a box to select multiple lines. 2. Then proceed to click the button the left main panel that says remove. 3. The line should then be removed.

isolating lines in the visualization

1. To isolate a lin,e click a line until it flashes, or drag a box to select multiple lines. 2. Then proceed to click the button the left main panel that says isolate. 3. That line should then be the only line displayed.

Showing all the lines in the visualization

Any lines that have been isolated or remved can be shown again. Simply click the show all button the left main panel, to see all the lines rendered.

Changing the color of lines in the visualization

Lots of Lines actually has a very robust line option system that can let you change the color of a line. 1. click or select multiple lines until the lines start to flash. 2. Once the lines are flashing, press the Line button on the left main panel. 3. This will bring a options menu pop out, in this menu, you will have three sliders that control the rgb values of color generated. To the left of those rgb sliders, there is a box that shows the current color being mixed. 4. Once you have the color you like, simply click apply and then okay, and the lines selected will that color.

Changing the width of lines in the visualization

To change the width of a line. 1. click or select multiple lines until the lines start to flash. 2. Once the lines are flashing, press the Line button on the left main panel. 3. This will bring a options menu pop out, in this menu, you will see a text box that says line width. 4. Enter a value into width between 0 and 1. 5. Then proceed to click apply and okay, then you should see the width of that line change.

Miscellaneous settings

Changing the line type to Arrows

Lots of lines has three different ways the actual lines can be shown in a visualization. 1. Click the bottom left gear icon. 2. A dialog box will the appear in the middle of the screen. 3. Simply click the button that says arrow and click okay, and you will now see that lines have arrows attached to them.

Changing the line type to cones

Lots of lines has three different ways the actual lines can be shown in a visualization. 1. Click the bottom left gear icon. 2. A dialog box will the appear in the middle of the screen. 3. Simply click the button that says cones and click okay, and you will now see that lines slowly become pointed, aka cones.

Changing the background color

Lots of lines has a system in place that allows it change the entire background color. 1. Click the bottom left gear icon. 2. A dialog box will appear in the middle of the screen. 3. At the top of the dialog box you will see three sliders, under background color, these sliders are for the rgb value of the colors. 4. Click okay to select the color for the background.

Changing the global line color

Lots of lines has a system in place that allows it change the global line color. 1. Click the bottom left gear icon. 2. A dialog box will appear in the middle of the screen. 3. At the top of the dialog box you will see three sliders, under Global line color, these sliders are for the rgb value of the colors. 4. Click okay to select the color for the lines.

Normalizing the Data

Lots of lines has a system in place that allows it to normalize the data. 1. Click the bottom left gear icon. 2. A dialog box will appear in the middle of the screen. 3. At the bottom of that dialog box, you will see a normalization text box, in that text box enter a number, you would like the data to be normalized by. 4. Then click okay to set that value.
