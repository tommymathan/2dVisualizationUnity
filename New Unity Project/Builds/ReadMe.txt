****Help documentation provided in the program by clicking “Help”****

To run the standalone data visualization application extract the contents of this zip file to your disk and run DVA.exe. 
To utilize the Excel plugin feature on Windows7, extract the contents of this zip file to your disk and open the DataVisualIzationPlugin folder.  Install the DVPcert, then run setup.exe from this directory to install the plugin. Run DVA.exe.  Next, use the provided DataVisualizationPlugin.xls file to push your dataset to the program by highlighting your desired data in Excel, select Addins, and click the button labelled “Export to Lots of Lines Application”. Maximize the Lots of Lines application and click “Ok” when you receive a popup asking you to utilize the new data pushed from Excel.
